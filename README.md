# Student Performance Dashboard - Nexus Analytics

## 🚀 Quick Start (Dash Backend)

1. **Install Python deps** (global or venv):
```
pip install dash pandas plotly numpy
```

2. **Run the dashboard**:
```
python app.py
```

3. **Open in browser**: http://127.0.0.1:8050

**Features**: Futuristic glassmorphism UI, sidebar filters, 5 animated charts, KPIs, responsive.

## 📱 Frontend (React/Vite)

Currently unused. Run:
```
npm install
npm run dev
```

## 📊 Dataset
Student Performance (UCI) - mat.csv loaded local or online fallback.

## Deployment
- Render/Heroku: Push repo, `pip install -r requirements.txt`
- Local CSV optional for offline.

Enjoy the premium analytics platform! 🎉
